--- HOW TO DEPLOY GAMITTER ---

1 - Execute MysqlServer or Wamp
2 - Execute the script of sql (epawtwitter.sql) that contains all our BD
3 - Open Eclipse
4 - Import gamitter.war
5 - Create server Apache Tomcat v8.0, with version of JDK "jdk1.8.045"
6 - Start the server
7 - access to http://localhost:8080/gamitter/main
8 - Enjoy!!
